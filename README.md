## :fire: Features

- Shorten URL's with a single click
- simple and responsive UI

- **NodeJS**
- **ExpressJS**

## Instructions to run

- Pre-requisites:

  - Node v12.0
  - Node Package manager (npm)

- To install all the dependencies

```bash
npm install
```

- To run the project

```bash
nmp start
```
